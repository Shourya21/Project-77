Console

The Web Console Logs information associated with a web page, for eg: network requests, JavaScript, CSS, security errors and warnings as well as error, warning and informational messages explicitly logged by JavaScript code running in the page context. JavaScript console log function is mainly used for code debugging as it makes the JavaScript print the output to the console.


To open the console in Chrome, use this keyboard shortcut: Cmd + Option + J (on a Mac) or Cmd +Shift +J (on Windows). As an alternative, you could right-click on the webpage and click "Inspect," and the developers window will appear. Click the "Console" tab in that window.
